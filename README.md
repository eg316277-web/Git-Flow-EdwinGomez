## 📝 Descripción
Este repositorio contiene la implementación lógica del sistema **CodeRider**. Se enfoca en el manejo profesional de ramas utilizando la metodología **Git Flow**, integrando pruebas unitarias y flujos de trabajo de desarrollo (Develop), lanzamientos (Release) y correcciones (Hotfix).

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** JavaScript (ES6+)
* **Entorno:** Node.js
* **Control de Versiones:** Git & Git Flow
* **Plataforma:** GitHub

## 📂 Estructura del Proyecto
* `src/app.js`: Lógica principal (saludos, sumas, estado del sistema).
* `test/app.test.js`: Scripts de prueba para validar el funcionamiento.
* `CHANGE.md`: Historial de cambios por versión.
